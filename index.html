All Features & Modules

14 powerful modules covering every aspect of school management — from admissions to payroll

🔐
Core

User & Authentication

Multi-role system with JWT token authentication. Role-based access control, password management, and user search.

✓ 4 user roles (Admin/Teacher/Student/Parent)
✓ JWT Access (1hr) + Refresh (7 days) tokens
✓ Admin can change any user's password
✓ UUID-based user IDs
📚
Core

Academic Structure

Manage classes, sections, groups, subjects, and weekly routines with ease.

✓ Class Levels (6-12) & Sections (Padma/Meghna/Jamuna)
✓ Groups: Science, Arts, Commerce
✓ Subjects with unique codes per class
✓ Weekly routine: day/period/teacher/room
👨‍🏫
HR

Teacher Management

Complete teacher lifecycle management with profiles, attendance, and class tracking.

✓ Auto-generated teacher ID with photo & NID
✓ Daily attendance (Present/Absent/Late/Leave)
✓ Class history recording per subject/topic
✓ PDF: ID Card & Appointment Letter
👩‍🎓
Core

Student Management

End-to-end student management with profiles, attendance tracking, and reporting.

✓ Auto student ID with personal & academic info
✓ Daily attendance (Present/Absent/Late/Holiday)
✓ Monthly attendance summary with %
✓ PDF: ID Card & Attendance Report
👨‍👩‍👧
Portal

Parent Portal

Real-time child tracking: attendance, results, fees, and direct teacher messaging.

✓ Dashboard with attendance, results & fee status
✓ Direct messaging with teachers (reply threads)
✓ Real-time read status & conversation history
✓ School notices & announcements
📝
Academic

Examination System

Complete exam lifecycle from creation to results with 4 PDF documents.

✓ Exam creation per class with academic year
✓ Grade system (A+ to F) with auto-calculation
✓ Class tests per subject/section
✓ PDF: Routine, Tabulation, Marksheet, Admit Card
💰
Finance

Fee Management

Flexible fee collection with multiple categories, payment methods, and receipts.

✓ Custom fee heads (Tuition, Bus, Lab, Library, etc.)
✓ Multi-item receipts with auto numbering
✓ Payment methods: Cash, Online, Cheque
✓ Due tracking & professional receipt PDF
📊
Finance

Accounts & Bookkeeping

Full accounting system with income/expense tracking and period-wise reports.

✓ Customizable income/expense heads
✓ Daily transaction recording
✓ Head-wise filtering & summaries
✓ PDF: Period-wise accounts report
📦
Operations

Inventory Management

Track products, stock levels, and distribution across the institution.

✓ Product categories (Books, Stationery, Lab, etc.)
✓ Stock tracking with units (pcs/kg/box)
✓ Distribution to class or individual
✓ Notes & date tracking
👔
HR

Staff Management

Manage non-teaching staff with departments, designations, and appointments.

✓ Departments: Admin, Accounts, Library, Lab, etc.
✓ Designations with department mapping
✓ Staff profiles with NID, salary, status
✓ PDF: Appointment Letter
💸
HR

Payroll Management

Unified payroll for teachers & staff with full salary structure and reporting.

✓ Bank details, allowances, deductions setup
✓ Leave types: Casual, Sick, Earned, Maternity, etc.
✓ Loan/advance salary with EMI tracking
✓ 3 PDFs: Payslip, Salary Statement, Collective
📢
Website

Content Management

Full public website with hero slides, notices, events, blogs, and more.

✓ Hero slides with bilingual (EN/BN) titles & CTAs
✓ Notices (5 categories), events with gallery
✓ FAQs, blogs, contact forms, event registration
✓ 60+ public pages
📱
Communication

Notifications & Messaging

Internal notifications, parent-teacher chat, and SMS/WhatsApp gateway.

✓ System-level internal notifications (read/unread)
✓ Parent-teacher messaging with image attachments
✓ Reply threads with full conversation history
✓ SMS/WhatsApp gateway logging
💼
HR

Recruitment System

Post jobs, accept applications, and manage hiring pipeline.

✓ Job postings with requirements & deadline
✓ Online applications with resume upload
✓ Status pipeline: Applied → Shortlisted → Selected
✓ Internal notes & admin panel
